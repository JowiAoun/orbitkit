## Props

The Mantine Button component accepts all the props from the native button element and the following props:

- `variant`: Defines the button's appearance ('filled', 'light', 'outline', 'transparent', 'white', 'subtle', 'default', 'gradient')
- `size`: Controls the button's size ('xs', 'sm', 'md', 'lg', 'xl')
- `color`: Sets the button's color theme
- `radius`: Controls the border-radius ('xs', 'sm', 'md', 'lg', 'xl')
- `loading`: Shows a loading spinner
- `loaderProps`: Customizes the loader component
- `leftSection`: Adds content before the button label
- `rightSection`: Adds content after the button label
- `fullWidth`: Makes the button take up 100% of container width
- `disabled`: Disables the button
- `gradient`: Configures the gradient when variant is set to 'gradient'

For more detailed documentation, visit the [Mantine Button documentation](https://mantine.dev/core/button).
